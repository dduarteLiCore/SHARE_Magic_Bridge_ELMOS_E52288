================================================================================
USO DE LOS ARCHIVOS
================================================================================

Para desarrolladores LabVIEW:

1. Lee README.md en labview_vis/
2. Implementa ejemplos en orden (1 → 2 → 3 → 4)
3. Usa scripts de bin/ para testing
4. Personaliza según necesidades

Para scripts Python (bin/):

detect_port.py:
  python3 detect_port.py
  # Output: "COM3" o "/dev/ttyACM0" o "NOT_FOUND"

magicbridge_api.py:
  python3 magicbridge_api.py load file.hmf COM3 5 D
  python3 magicbridge_api.py cmd COM3 W5d
  # Output: JSON estructurado

Wrappers:
  ./magicbridge-load file.hmf /dev/ttyACM0 5 D --json out.json
  ./magicbridge-cmd /dev/ttyACM0 W5d --json write.json
